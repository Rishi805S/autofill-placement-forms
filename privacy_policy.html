Privacy Policy — Placement Form Autofill

Last updated: October 13, 2025

Summary

This extension stores user profiles locally in the browser (using chrome.storage.local) and does not transmit profile data to any external servers by default.

What we collect

No data is collected by us. All profile data (name, email, phone, education, resume link, preferences such as relocation and gender) is stored locally on your device and accessible by the extension only.

Local storage

Profiles are stored under the key profilesV1 in chrome.storage.local. You can export profiles as JSON files manually and import them back. Deleting a profile in the extension removes it from local storage.

Cloud sync (future / opt-in only)

There is no cloud sync feature in the current release. If cloud sync is added later, it will be strictly opt-in and the privacy policy will be updated to describe what is shared, how it is secured, and how to opt out.

Third-party services

This extension does not call third-party APIs or analytics services in the current release.

Contact

If you have questions about privacy or data handling, please open an issue in the repository or contact the developer at: dev@example.com.
